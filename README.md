# TUNIA - Tunisian AI Academic Assistant

TUNIA is a beautiful AI chat website built with React and Bootstrap, designed specifically for Tunisian students to ask academic questions in Tunisian Darija (dialect). The AI responds exclusively in Tunisian Darija, making academic help more accessible to local students.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/tunia-chatbot.git
cd tunia-chatbot
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## Usage

1. **Welcome Screen**: Start with an attractive welcome screen showcasing TUNIA's capabilities
2. **Chat Interface**: Ask questions in Tunisian Darija about academic topics
3. **PDF Upload**: Upload PDF files for document analysis and questions
4. **Real-time Responses**: Get instant responses from the AI assistant
